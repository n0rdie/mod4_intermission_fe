### About
Users search for most recent news and are given brief synopses of the article.

### Setup
- Clone this repo
- `bundle install`
- `rails s`
- Go to localhost:3000

### API
This project consumes https://newsapi.org

### Tests
`bundle exec rspec`
- `news_facade_spec`
- `artical_poro_spec`
- `news_index_spec` (many articles)
- `news_index_spec` (one article)
